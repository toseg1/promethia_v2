⚙️ Server Management

Starting the Backend Server (Django)

# Navigate to backend directory cd /Users/Gins/developments/promethia_react_v2/backend # Start development server python manage.py runserver --settings=backend.settings_dev # Alternative: Start on specific port python manage.py runserver 8000 --settings=backend.settings_dev # Run in background (optional) nohup python manage.py runserver --settings=backend.settings_dev &
✅Backend Server: Currently Running on http://localhost:8000

Starting the Frontend Server (React + Vite)

# Navigate to frontend directory cd /Users/Gins/developments/promethia_react_v2/frontend # Install dependencies (first time only) npm install # Start development server npm run dev # Alternative: Start on specific port npm run dev -- --port 3000 # Build for production npm run build # Preview production build npm run preview
⚠️Frontend Server: Not Running - Use npm run dev to start on http://localhost:5173

Stopping Servers

# Stop servers running in terminal Ctrl + C # Kill server by port (if running in background) lsof -ti:8000 | xargs kill -9 # Kill Django server lsof -ti:5173 | xargs kill -9 # Kill Vite server # Find processes using ports lsof -i :8000 # Check Django port lsof -i :5173 # Check Vite port

🔧 Troubleshooting

Common Issues & Solutions

🚨 "Port already in use" Error

# Find and kill process using port lsof -ti:8000 | xargs kill -9 # Kill Django server lsof -ti:5173 | xargs kill -9 # Kill Vite server # Or use different ports python manage.py runserver 8001 # Different port for Django npm run dev -- --port 3000 # Different port for Vite

🚨 Database Migration Issues

# Check migration status python manage.py showmigrations # Reset migrations (CAUTION: loses data) rm -rf accounts/migrations/00*.py rm -rf core/migrations/00*.py python manage.py makemigrations accounts core python manage.py migrate # Or fake apply specific migration python manage.py migrate accounts 0001 --fake

🚨 Frontend Build Issues

# Clear node modules and reinstall rm -rf node_modules package-lock.json npm install # Clear Vite cache rm -rf node_modules/.vite # Update dependencies npm update

🚨 CORS Issues

# Check Django settings_dev.py for: CORS_ALLOWED_ORIGINS = [ "http://localhost:3000", "http://localhost:5173", ] # Or temporarily allow all (NOT for production) CORS_ALLOW_ALL_ORIGINS = True

Health Check Commands

# Backend health checks curl http://localhost:8000/admin/ # Should return Django admin page python manage.py check # Django system check python manage.py check --deploy # Production deployment check # Frontend health checks curl http://localhost:5173/ # Should return frontend npm run type-check # TypeScript checks npm run lint # Code style checks # Database checks python manage.py showmigrations # Migration status sqlite3 backend/db.sqlite3 ".tables" # List database tables
⚠️Important: Always backup your database before making major changes. The database file is located at backend/db.sqlite3
